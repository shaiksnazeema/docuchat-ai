# 📄 DocuChat AI

An intelligent PDF chatbot built using Python and Streamlit that allows users to upload PDF documents, extract text, and interact with their content efficiently.

## 🚀 Features

- Upload PDF documents
- Extract text instantly
- Interactive web interface
- Built with Python and Streamlit
- Clean and user-friendly UI
- Easily extendable for AI-powered question answering

## 🛠️ Technologies Used

- Python
- Streamlit
- PyPDF2
- Python Dotenv
- LangChain
- OpenAI
- FAISS
