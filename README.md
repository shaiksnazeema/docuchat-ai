 🚀 DocuChat AI — Intelligent PDF Question Answering System
> Transform static PDF documents into interactive conversations using AI.
DocuChat AI is a smart document assistant built with Python, Streamlit, and modern NLP frameworks. It enables users to upload PDF files, extract textual content, and interact with documents through an intuitive web interface. This project demonstrates practical skills in document processing, natural language understanding, and full-stack AI application development.

 🌟 Key Features

 📄 Upload and process PDF documents instantly
 🔍 Extract and analyze text from multi-page PDFs
 💬 Interactive chatbot-style interface
⚡ Fast and responsive Streamlit web application
 🧠 Designed for seamless integration with LLMs
 🔐 Secure environment variable management
📈 Scalable architecture for future enhancements

🛠️ Tech Stack
* Programming Language: Python
* Frontend: Streamlit
* PDF Processing:PyPDF2
* Environment Management: Python Dotenv
* AI Framework: LangChain
* Vector Database: FAISS
* LLM Integration: OpenAI API



🏗️ System Architecture

User Uploads PDF
       │
       ▼
PDF Text Extraction (PyPDF2)
       │
       ▼
Text Processing & Chunking
       │
       ▼
Vector Embeddings (FAISS)
       │
       ▼
LLM Query Processing
       │
       ▼
Interactive Response via Streamlit

 ⚙️ Installation Guide

bash
# Clone the repository
git clone https://github.com/shaiksnazeema/docuchat-ai.git

# Navigate to project directory
cd docuchat-ai

# Create virtual environment
python -m venv venv

# Activate virtual environment
# Windows
venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Run the application
streamlit run app.p
